2. ฟังก์ชันหลักของ M2 (2 ส่วน)
ส่วนที่ 1: เก็บค่าเช่าจากร้านค้าในปั๊ม
ฟังก์ชันรายละเอียดตัวอย่าง1. สัญญาเช่าพื้นที่บันทึกสัญญาแต่ละร้าน (เริ่ม-สิ้นสุด, อัตราค่าเช่า)FIT Auto: 7,000 บาท/เดือน2. คำนวณค่าเช่าอัตโนมัติแยกตามประเภท:
• ค่าเช่าคงที่
• % จากยอดขายChester’s: 5% จากยอดขาย3. ออกใบแจ้งหนี้สร้างใบแจ้งหนี้ PDF รายเดือน“ใบแจ้งหนี้ค่าเช่า ต.ค. 2568”4. บันทึกการชำระเงินเงินสด / โอน / หักจากยอดขายรับเงิน 7,000 บาท วันที่ 55. ติดตามหนี้ค้างชำระแจ้งเตือน 7/15/30 วัน“Daiso ค้าง 1 เดือน”6. รายงานรายได้ค่าเช่ารายปั๊ม / รายเดือน / รายร้าน“รวมค่าเช่าทั้ง 5 ปั๊ม: 150,000 บาท”
ส่วนที่ 2: ค่าเช่าที่ปั๊มไปเช่าที่อื่น
ฟังก์ชันรายละเอียดตัวอย่าง1. บันทึกค่าเช่าภายนอกปั๊มเช่าที่ดิน/ป้าย/โฆษณาเช่าที่ดิน 20,000 บาท/เดือน2. ออกใบสำคัญจ่ายสร้างเอกสารจ่ายเงิน“จ่ายค่าเช่าที่ดิน 20,000 บาท”3. ติดตามกำหนดชำระแจ้งเตือนก่อนครบกำหนด“ครบกำหนด 25 พ.ย.”4. รายงานค่าใช้จ่ายแยกตามปั๊ม“ค่าเช่าภายนอกทั้งหมด: 50,000 บาท”
3. เชื่อมต่อกับโมดูลอื่น (สำคัญมาก!)
โมดูลการเชื่อมต่อM1ดึงยอดขายร้าน (เช่น Chester’s) → คำนวณ % ค่าเช่าM6 (บัญชี)ส่งค่าเช่าเข้า → รายได้ (4110) / ค่าใช้จ่าย (5120)M7 (แดชบอร์ด)แสดง “รายได้ค่าเช่า vs ค่าใช้จ่ายเช่า”
4. ตัวอย่างการใช้งานจริง
ขั้นตอนการทำงาน1. สร้างสัญญากรอก: FIT Auto สาขา A → 7,000 บาท/เดือน2. ดึงยอดขายจาก M1 → Chester’s ขาย 500,000 บาท3. คำนวณค่าเช่า5% = 25,000 บาท4. ออกใบแจ้งหนี้ส่งให้ Chester’s5. รับเงิน25,000 บาท → บันทึก6. ลงบัญชีM6: เดบิต เงินสด, เครดิต รายได้ค่าเช่า

6. ข้อดีของโมดูล M2
ข้อดีประโยชน์ลดงานมือออกใบแจ้งหนี้ + คำนวณอัตโนมัติติดตามหนี้ได้แจ้งเตือนทันทีวิเคราะห์ได้รู้ว่าร้านไหนทำกำไรค่าเช่ามากสุดเชื่อมบัญชีลงบัญชีอัตโนมัติ (M6)ครอบคลุม 2 ทางทั้ง “เก็บ” และ “จ่าย”
7. ข้อเสนอแนะเพิ่มเติม (เพื่อความสมบูรณ์)
ฟีเจอร์งบเพิ่มเหตุผลOCR อ่านสัญญาเช่า+4,000 บาทสแกนสัญญา → ลดกรอกมือแจ้งเตือน Line+3,000 บาทแจ้งหนี้ค้างให้ผู้จัดการคำนวณ % ยอดขายอัตโนมัติ+5,000 บาทดึงจาก M1 โดยตรงพิมพ์ใบเสร็จรับเงิน+2,000 บาทสำหรับร้านที่จ่ายเงินสด
รวมงบเพิ่ม: +14,000 บาท → M2 กลายเป็น 84,000 บาท
สรุปสั้น ๆ (1 บรรทัด)
M2 = จัดการ “ค่าเช่าที่เก็บจากร้านในปั๊ม” + “ค่าเช่าที่ปั๊มจ่ายออก” → ออกใบแจ้งหนี้, ติดตามหนี้, ลงบัญชีอัตโนมัติ (70,000 บาท)
